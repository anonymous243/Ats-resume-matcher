# ATS Resume Matcher

## Project Overview

The ATS Resume Matcher is designed to match candidates' resumes with job descriptions, helping recruiters and hiring managers streamline their selection processes. By leveraging natural language processing (NLP) techniques, the application can identify key skills and qualifications in resumes and compare them with the requirements listed in job postings.

## Features

- **Resume Parsing:** Parse resumes to extract relevant information such as education, work experience, and skills.
- **Job Description Analysis:** Analyze job postings to identify key qualifications and skills required.
- **Matching Algorithm:** Calculate the match percentage between resumes and job descriptions based on extracted data.
- **User-Friendly Interface:** Provide an intuitive UI for both job seekers and recruiters.
- **Performance Metrics:** Offer insights into the effectiveness of resumes in matching job descriptions.

## Installation

To install the project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/anonymous243/Ats-resume-matcher.git
   cd Ats-resume-matcher
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```

## Usage

1. Upload the candidate's resume.
2. Enter the job description to be matched.
3. Click the "Match" button to see the results.
